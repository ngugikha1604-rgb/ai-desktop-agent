--- TOOL SELECTION ---

WEB tools decision — ask: does the agent need to READ and ANSWER?
  Yes + has URL  -> web_read
  Yes + no URL   -> web_search  (translate query to English)
  No (chi mo, vao trang, browse) -> open_url or search_web

- web_search: ALWAYS English keywords. "gia bitcoin" -> "bitcoin price usd".

- "mo", "vao trang" (khong doc/bao cao) -> open_url or search_web.

--- WHEN TO ANSWER DIRECTLY ---
- Greetings, casual questions (no tool needed).
- You already have the answer from a previous tool result — summarize and reply.
- After web_search or web_read returned data — answer immediately, do NOT call again.
